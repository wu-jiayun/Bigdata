# 湖南省农业种植适宜性区划分析系统

基于大数据技术的智能农业分析平台，专注于湖南省农业种植适宜性区划分析与联网报告生成。

## 项目概述

本系统是一个集成多准则评价、实时数据分析和智能报告生成的现代化农业分析平台。系统整合了气候、土壤、地形等多维度数据，结合实时联网信息，为湖南省农业种植决策提供科学依据和智能建议。

### 核心特性

- **联网数据集成**: 实时获取市场行情、天气预报、政策动态
- **多准则区划分析**: 支持12种农作物的适宜性评价
- **可视化展示**: 基于湖南省地图的交互式数据展示
- **智能报告生成**: 自动生成包含联网数据的完整分析报告
- **完整数据导出**: 支持纯文本格式的完整数据导出

### 主要功能模块

#### 1. 多准则适宜性区划

- **作物类型**: 水稻、玉米、大豆、小麦、棉花、油菜、花生、红薯、烟草、茶叶、柑橘、蔬菜
- **区划精度**: 县级、乡镇级、村级三种精度
- **评价指标**: 温度适宜度、土壤适宜度、水分适宜度
- **可视化**: 湖南省地图热力图展示

#### 2. 联网报告生成

- **实时市场数据**: 当前价格、价格趋势、市场需求
- **天气预报集成**: 温度范围、降水预测、种植条件
- **政策动态跟踪**: 最新补贴政策、产业规划、指导意见
- **技术趋势分析**: 智能化种植、新品种推广、防控技术
- **产业分析**: 种植面积、预期产量、市场前景

#### 3. 数据分析与可视化

- **温度分析**: 年度和月度温度趋势分析
- **土壤分析**: 土壤类型分布和肥力评价
- **作物分析**: 多作物适宜性对比评价
- **交互式图表**: ECharts图表库支持的动态可视化

#### 4. 完整数据管理

- **数据预览**: 报告预览中展示所有分析数据
- **县市详情**: 14个地级市的详细评分和等级
- **数据导出**: 纯文本格式完整报告下载
- **数据整合**: 本地分析数据与联网实时数据融合

## 快速开始

### 环境要求

- Python 3.8+
- MySQL 5.7+
- 8GB+ RAM（推荐）
- Apache Spark（用于大数据处理）

### 安装步骤

1. **克隆项目**

```bash
git clone [项目地址]
cd 湖南农业分析系统
```

2. **安装依赖**

```bash
pip install -r requirements.txt
```

3. **配置数据库（可选）**

- 编辑 `config.py` 文件设置MySQL连接参数
- 系统支持模拟数据模式，无需数据库即可运行

4. **启动应用**

```bash
python app.py
```

5. **访问系统**

- 打开浏览器访问：http://localhost:5004
- 系统自动初始化，支持模拟数据和真实数据两种模式
- 通过主仪表板访问各功能模块

## 🚀 使用指南

### 主要功能使用

1. **多准则适宜性区划**
   - 访问主仪表板，选择"多准则适宜性区划"
   - 选择作物类型（12种作物可选）
   - 查看湖南省地图可视化结果
   - 分析适宜性等级分布

2. **联网报告生成**
   - 进入"报告生成"模块
   - 选择作物类型和报告标题
   - 系统自动获取联网实时数据
   - 预览完整报告内容
   - 下载纯文本格式完整报告

3. **数据分析模块**
   - **温度分析**: 查看年度和月度温度趋势
   - **土壤分析**: 分析土壤类型分布和适宜性
   - **作物分析**: 对比不同作物的适宜性评价
   - **区划分析**: 查看详细的区划统计结果
