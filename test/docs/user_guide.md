# 用户使用指南

## 系统概述

湖南省基于Spark的农业种植适宜性区划与优化系统是一个专业的农业数据分析平台，通过分析气候、土壤、作物等多维度数据，为农业决策提供科学依据。

## 功能模块

### 🌡️ 气候趋势分析
分析湖南省气候变化趋势，包括：
- **月度温度变化趋势**：展示全年12个月的温度变化规律
- **年度温度趋势**：多年温度数据对比分析
- **季节温度对比**：春夏秋冬四季温度分布统计

### 🌱 土壤分布分析
分析湖南省土壤特征分布，包括：
- **土壤类型分布**：8种主要土壤类型的地理分布
- **pH值分布**：土壤酸碱性分布统计
- **县市土壤质量排名**：13个县市的土壤质量对比

### 🌾 作物适宜性分析
分析作物种植适宜性，包括：
- **作物分类分布**：6大类61种作物的分布情况
- **作物温度需求**：不同作物的温度适应范围
- **作物pH适应性**：作物对土壤酸碱性的适应能力

### 🗺️ 区划优化建议
提供种植区划优化方案，包括：
- **区划优化分布**：基于多因子分析的优化建议
- **最佳种植区域**：推荐的最适宜种植区域

## 操作指南

### 第一步：访问系统
1. 打开浏览器（推荐Chrome、Firefox、Safari）
2. 访问系统地址：http://localhost:5004
3. 等待页面完全加载

### 第二步：初始化系统
1. 点击**"初始化Spark系统"**按钮
2. 等待系统连接MySQL数据库
3. 查看数据摘要信息：
   - 作物类型数量
   - 降水记录数量
   - 土壤记录数量
   - 温度记录数量

### 第三步：运行分析
1. 点击**"运行综合分析"**按钮
2. 系统开始处理农业数据
3. 等待分析完成提示
4. 图表自动加载显示

### 第四步：查看结果
1. 使用顶部标签页切换不同分析类别
2. 每个标签页包含3-4个专业图表
3. 图表支持交互操作：
   - 鼠标悬停查看详细数据
   - 点击图例控制数据系列显示/隐藏
   - 使用工具栏进行缩放、保存等操作

## 界面说明

### 导航栏
- **系统标题**：显示系统名称和图标
- **系统状态**：实时显示系统运行状态
  - 🟡 未初始化
  - 🟡 初始化中/分析中
  - 🟢 系统就绪/分析完成
  - 🔴 错误状态

### 控制面板
- **初始化按钮**：连接数据库，初始化系统
- **分析按钮**：运行数据分析（需先初始化）
- **状态信息**：显示操作结果和系统信息

### 分析结果区域
- **标签页导航**：4个分析类别的切换标签
- **图表展示**：每个类别包含多个专业图表
- **响应式布局**：自动适配不同屏幕尺寸

## 图表操作

### 基本操作
- **查看数据**：鼠标悬停在图表元素上查看详细数值
- **缩放**：使用鼠标滚轮或工具栏进行缩放
- **平移**：按住鼠标左键拖拽图表
- **重置**：双击图表恢复原始视图

### 图例控制
- **显示/隐藏**：点击图例项控制对应数据系列
- **高亮**：鼠标悬停在图例上高亮对应数据

### 工具栏功能
- **保存图片**：将图表保存为PNG格式
- **数据视图**：查看图表的原始数据
- **配置项**：调整图表显示参数
- **刷新**：重新加载图表数据

## 常见问题

### Q1：图表不显示怎么办？
**A1**：请检查以下几点：
- 确保已完成系统初始化
- 确保已运行综合分析
- 尝试切换到其他标签页再切换回来
- 刷新浏览器页面

### Q2：分析时间过长怎么办？
**A2**：分析时间取决于数据量和系统性能：
- 正常情况下需要30-60秒
- 如超过2分钟，请检查数据库连接
- 可以查看浏览器控制台了解详细状态

### Q3：如何保存分析结果？
**A3**：可以通过以下方式保存：
- 使用图表工具栏的保存功能
- 浏览器截图保存
- 打印页面为PDF

### Q4：支持哪些浏览器？
**A4**：推荐使用现代浏览器：
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### Q5：数据更新频率如何？
**A5**：
- 基础数据相对稳定（土壤、地形等）
- 气候数据可定期更新
- 作物数据根据农业发展调整
