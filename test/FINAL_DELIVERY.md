## 🚀 部署和运行

### 系统要求
- **Python**: 3.8或更高版本
- **MySQL**: 5.7或更高版本
- **内存**: 8GB以上(推荐16GB)
- **存储**: 20GB可用空间

### 快速启动
```bash

pip install flask pymysql numpy pandas

# 2. 配置数据库
# 修改config.py中的数据库连接信息

# 3. 启动应用
python app.py

# 4. 访问系统
# http://localhost:5004
```

### 生产部署
```bash
# 使用Gunicorn
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:5004 app:app
```
